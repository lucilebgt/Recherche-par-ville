# Recherche-par-ville
app permettant de trouver une ville de différentes façons :

    - par nom,
    - par code postal,
    - par code departement,


*Api*:

    - geo gouv : geo.api.gouv.fr
    - meteo: OpenWeather 

*technos*:

    -React, JSX, Jest, Parcel
